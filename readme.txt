Запускать приложение на dev-окружении с помощью команды: npm start;

Для сборки продакшн-версии используйте команду: npm run build;

Запустить JSON-server с помощью команды: npm run server;

Автоматически пересобирать проект при изменении исходных файлов в режиме разработки, запустить webpack-dev-server: npm run dev
